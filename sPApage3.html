softPetroleum Appraiser™

Property Tax Valuation Software

softPetroleum Appraiser™ (sPA) provides specialized tools for determining property tax value of oil and gas properties. sPA includes a comprehensive data layer which contains well and lease historical data. Property biography, production, prices, expenses, division of interest are a few of the categories.

For Kansas properties, softPetroleum Appraiser provides specialized screens for calculating property tax value of oil and gas properties using the Kansas Oil and Gas Appraisal Guide method as well as alternative market approaches.

Historical information required for reporting can be entered or automatically loaded into the data layer. softPetroleum Appraiser retrieves and displays required data in the property tax interface. Required information can also be directly edited using the property tax data tabs.

The Kansas property tax interfaces are also linked to softPetroleum Appraiser's decline curve analysis / discounted cash flow analysis tools. Users can compare results using multiple methods such as decline–curve based in–place reserves value.

Calculations for the oil and gas rendition forms can be viewed from the Kansas property tax interface. Modifications to the proscribed Guide calculation can be submitted using the "Owner" opinion of value column.

Adjusted values can be entered into the "Owner" or "Appraiser" column as appropriate or automatically generated based on alternative valuation calculations. softPetroleum Appraiser™ provides columns for the proscribed Schedule valuation, Owner's opinion of value and the Appraiser's final value.
